## Functional Programming in Scala - Second Edition

### Usage

This project is built using `scala-cli`.

Compile the project:

```bash
scala-cli compile .
```

To test a specific chapter, run:

```bash
scala-cli test . --test-only il.co.dotcore.julian.fpinscala2023.ch02.\*
```

Run the repl:

```bash
scala-cli repl .
```
